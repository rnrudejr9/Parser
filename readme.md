# LineParser

* IDE : intellij
* Language : java
* Database : mysql(jdbc driver gradle)
* Server : aws
* 프로그램
    * csv파일을 읽어 필요한 부분을 추출
    * 추출한 내용을 객체에 저장
    * 객체 내용을 DB에 연동해서 Insert
    * DB연동해서 작업 추가 
    * 테스트 코드 활용
